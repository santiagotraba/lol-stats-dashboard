# LoL Stats Dashboard

A modern web application that allows users to explore and analyze player statistics from League of Legends.  
The dashboard provides insights into player performance, champion usage, match history, and gameplay trends through a clean and responsive interface.

## Live Demo

https://lol-stats-dashboard.vercel.app

## Repository

https://github.com/santiagotraba/lol-stats-dashboard

## Features

- Search for League of Legends summoners
- View player profile information
- Champion performance statistics
- Match history overview
- Interactive charts and data visualization
- Responsive and modern dashboard UI

## Tech Stack

This project was built with:

- React
- TypeScript
- Vite
- Tailwind CSS
- TanStack Query
- Recharts
